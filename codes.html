Multi-language School Assignment Algorithms

A public GitHub repository can be found here.

The School Assignment Algorithms (SAA) package offers a flexible set of functions for implementing student-to-school assignment mechanisms based on a wide range of rules, priorities, and quota specifications. This package aims to be a user-friendly tool for implementing student-to-school assignment mechanisms, while providing consistent results across various programming languages ​​(Python, R, and MatLab).

The repositoriy currently includes stable R version. A testing Python version can also be found there. The package currently includes:

  • Base student-to-school matching mechanism (Boston, DA or hybrid).
  • Prioritization of students based on N number of priority groups.
  • Hard quotas (no soft quotas supported in this version).
  • Admission probabilites statistics.
  • Waitlist statistics.
  • Cutoffs statistics.
  • Seats statistics.

A stable version of Python is expected to be released soon. Improvements such as group application and flexible quotas are planned for later.
